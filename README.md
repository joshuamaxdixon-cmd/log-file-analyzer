# Log File Analyzer

This project is a simple Python tool that scans server log files and detects errors.

## Features

- Reads log files
- Identifies error messages
- Counts total errors

## Technologies

Python

## How to Run

Run the analyzer with a log file:

python log_analyzer.py sample_log.txt

## Example

Sample log file:

sample_log.txt

Run the analyzer:

python log_analyzer.py sample_log.txt

## Example Output

Error found: Database connection failed
Error found: Timeout while contacting API

Total errors detected: 2
